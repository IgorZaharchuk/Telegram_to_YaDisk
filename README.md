# Telegram MTProto Backup to Yandex Disk

Автоматическое резервное копирование файлов из Telegram на Яндекс.Диск с поддержкой тем (topics).

## 🌟 Возможности

- ✅ **MTProto** — скачивание файлов любого размера (до 2GB, без лимита 20MB!)
- ✅ **Человеческие названия тем** — папки называются как темы в Telegram
- ✅ **Сжатие фото и видео** — экономия места на диске
- ✅ **Прогресс** — продолжает с того места, где остановился
- ✅ **GitHub Actions** — автоматический запуск по расписанию

## 🚀 Быстрый старт

### 1. Получите API credentials
1. Перейдите на https://my.telegram.org/apps
2. Войдите под своим номером телефона
3. Создайте приложение, получите `api_id` и `api_hash`

### 2. Настройте проект

cd tg-mtproto-backup
